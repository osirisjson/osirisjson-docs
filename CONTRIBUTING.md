# Contributing to OSIRIS JSON Documentation

We welcome and encourage everyone to contribute to the OSIRIS JSON Documentation repository!

Before submitting a pull request, please review our [Contribution Guidelines](https://docs.osirisjson.org/community/contributing/).

For details on grammar, formatting, and frontmatter requirements, consult the [OSIRIS JSON Style Guide](https://docs.osirisjson.org/community/style-guidelines/introduction/). You can also browse our guide on [using components](https://docs.osirisjson.org/community/style-guidelines/components/) to enhance your formatting.

Thank you for helping us maintain high-quality, accurate documentation.

## Our Policy on AI-Generated Content

While we fully support using AI tools for content validation, grammar checking, and typo fixes, **we require the core documentation to be authored by humans.** Please do not submit pull requests containing documentation generated entirely from scratch by Large Language Models.
