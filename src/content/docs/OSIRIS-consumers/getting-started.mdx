OSIRIS JSON Consumers are lightweight, on-premise application designed to close the circle of OSIRIS JSON ecosystem.
If you generate your infrastructure data with OSIRIS JSON Producers into an open, portable format, you should expect the resulting outputs to be equally accessible and vendor-neutral.

Consumers natively ingest your OSIRIS JSON documents to automate the tedious parts of infrastructure management:

*   **Visual Topologies:** Automatically generate and update architecture diagrams using [draw.io](https://www.drawio.com/).
*   **Native Documentation:** Generate comprehensive infrastructure reports.
*   **Drift Detection:** Run a `diff` between historical documents to instantly visualize configuration drift over time.
